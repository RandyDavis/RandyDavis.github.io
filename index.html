HTML


I specialize in writing clean, semantic, and maintainable HTML compliant code. HTML5 features are also used in my work on a daily basis. If you have any markup needs, then I can deliver them. Have a look at some of my past projects and see for yourself.

Responsive Web Design


I don't work on anything unless the end result is going to be responsive. It's how I learned and it's just the way to go with so many devices and varying screen sizes in this day and age. JavaScript/jQuery
